Installation

Installation is simple. Just download the zip file from Codecanyon and upload it to your website via Wordpress Plugins page.

Team Members

After the plugin activation, you will see a new menu item on the left side of your wordpress admin panel (DV-Galleries).

DV-Team: All your team members will be displayed on this page.

Add new member: You can use this menu item to add new members

Categories: You can use categories to organize your team members.

Main Screen
Categories Screen

Adding a new team member

Post title should contain member name.

You can add any content to the text editor like on the other Wordpress posts and pages.

Most important field is "Featured Image". Members without featured images will not be displayed on your pages.

There are 6 different format. As default "Standard" format is selected. If you don't change this, following post format fields will be disabled.

Post Formats

Image: You can use this format to add a single image to the top of the member page.

Gallery: You can use this format to add a slider to the top of the member page. After clicking "Add or Upload File" button, you can make a multiselection with CTRL+ click. After this selection you can drag-drop thumbnails to change their orders.

Video: You can copy paste your embed video code from any video sharing website like YouTube, Vimeo etc.

Quote: You can use this format to add an attractive quote to the top of the member page.

Link: If you don't want to use sliding panel for this member, you can select "Link" post format and add any website url to this field. Correct link format should be like the following;

http://www.themeforest.net

Social Icons

If you want to add a social icons bar to the left side of sliding member panel, first of all you should check "Activate Social Icons" box. After that you can add as many icon as you want. You can select an icon from the dropdown menu or you can upload your own icon. (If you add a custom icon, don't forget to change custom icon background color.) You should click "Add Another Icon" button to add a new icon.

Categories

You can add new categories directly from add/edit member page or from categories page like on the standard Wordpress posts. Categories may be useful if you want to create multiple teams.

Shortcodes

DV Team Shortcode

You can use this shortcode to add your team members to any post or page.

max: Maximum number of members.

categoryid: If you want to display team members from a specific category, you can add your category id/ids. (You can find category ids on DV-Team->Categories page);

gridstyle: There are 3 available option; full, square and rectangle. If you select "full", your featured images will be displayed as they are. To use square or rectangle thumbnail options, you should use images which you uploaded after the plugin activation. If you want to use images already existing in your media library, you can use "Regenerate Thumbnails" plugin. To download this plugin click here...

offset: You can change distance between thumbnails in the grid (px). You have to use a numeric value like 10,20,30 etc.

itemwidth: Minimum Item Width. You can play with this setting to create 1,2,3,4 etc. columns which fits your theme's design.

side: Sliding panel side: left , center or right

DV Member Shortcode

You can use this shortcode to add single member to any post or page. You can find member shortcodes on DV Team main page. Options are similar with Dv Team shortcode.

id: Every member has a unique id. If you use dvmember shortcode to add a single member, you don't need to edit this value. If you want to use this shortcode to add multiple members, you should edit id value and add member ids manually. You should also add commas between ids like on the following example;

[dvmember id="79,65,42" gridstyle="full" offset="20" itemwidth="250" side="right"]

To use square or rectangle thumbnail options, you should use images which you uploaded after the plugin activation. Widgets

There are 5 custom widget in DV-Team. If your theme supports widgets, you can use these widgets to add your your team members to your theme's widget fields.

If your theme doesn't supports widgets, you can use SiteOrigin Page builder to use widgets on any page. Siteorigin page builder is compatible with most of the themes but if you have any problem with this plugin, please contact your theme's author. There may be compability issues on some themes...

Options are the same as shortcodes.
